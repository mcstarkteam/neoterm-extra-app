NeoTerm-Extra

现代设计的 21 世纪安卓终端模拟器。

### 我们的承诺

最初 NeoTerm 被设计为 Termux 的前端，提供一些 Termux 没有的功能，但是我们
觉得很方便。 在不断的发展中，我们发现了我们的目标：做最好的安卓终端。
